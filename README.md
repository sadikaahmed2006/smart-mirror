# Smart Mirror Dashboard

A simple smart mirror dashboard prototype built using HTML, CSS, and JavaScript. This project is designed to run on a laptop browser and can later be adapted for a Raspberry Pi or mini PC smart mirror setup.

## Features

* Real-time digital clock
* Current date display
* Weather section placeholder
* Daily reminder section
* Quote section
* Minimal black-and-white smart mirror style interface
* Fullscreen browser display support

## Technologies Used

* HTML
* CSS
* JavaScript

## Project Purpose

The goal of this project is to build a software-only smart mirror prototype before moving to physical hardware. The laptop acts as the display for now, while the dashboard layout and functionality can later be transferred to a Raspberry Pi or other compact computer.

## How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Double-click `index.html`.
4. The smart mirror dashboard will open in your browser.
5. Press `F11` to view it in fullscreen mode.
